# WorkShift Calendar - Home Assistant Add-on

A powerful shift calendar for managing work schedules with Home Assistant integration.

## About

WorkShift Calendar allows you to:
- Manage work shifts with an intuitive calendar interface
- Create reusable shift templates for quick scheduling
- Integrate with Home Assistant via ICS feeds, REST API, and webhooks
- Run in admin mode to view and manage multiple users' calendars

## Installation

1. Add this repository to your Home Assistant Supervisor
2. Install the "WorkShift Calendar" add-on
3. Start the add-on
4. Click "OPEN WEB UI" to access the calendar

## Features

- Click-to-select workflow for placing shifts
- Maximum 2 shifts per day with horizontal split coloring
- Day notes tied to calendar days
- ICS feed for Home Assistant calendar integration
- REST API for automation
- Webhook support for event-driven updates
- Multi-user support with admin mode
