Home Assistant Integration
Remote Calendar Integration

Add your shift calendar to Home Assistant using the ICS feed URL above. 

# configuration.yaml
# Go to Settings > Integrations > Add Integration
# Search for "Remote Calendar"
# Enter your ICS feed URL
REST API Usage

Use the REST API for more advanced integrations with automations. 

# Example: Get events
GET http://your-server/api/v1/calendar/{api_key}/events?start=2024-01-01&end=2024-01-31

# Example: Create event
POST http://your-server/api/v1/calendar/{api_key}/events
{
  "summary": "Morning Shift",
  "start": "2024-01-15T08:00:00",
  "end": "2024-01-15T16:00:00"
}
Webhook Integration

Send events to your calendar from Home Assistant automations. 

# Example webhook payload
POST http://your-server/webhook/{api_key}
{
  "action": "create",
  "summary": "Shift Created via Webhook",
  "start": "2024-01-15T08:00:00",
  "end": "2024-01-15T16:00:00"
}
API Documentation
Available Endpoints
Endpoint Method Description
/ics/{api_key}.ics GET ICS feed for calendar integration
/api/v1/calendar/{api_key}/events GET List events (optional: ?start=&end=)
/api/v1/calendar/{api_key}/events POST Create a new event
/api/v1/calendar/{api_key}/events/{id} GET Get specific event
/api/v1/calendar/{api_key}/events/{id} PUT Update an event
/api/v1/calendar/{api_key}/events/{id} DELETE Delete an event
/webhook/{api_key} POST Webhook (action: create/delete)
Event Object Structure
{
  "id": 1,
  "summary": "Morning Shift",
  "start": "2024-01-15T08:00:00",
  "end": "2024-01-15T16:00:00",
  "description": "Optional day note",
  "color": "#3788d8"
}
Webhook Actions
# Create event
{"action": "create", "summary": "Shift", "start": "...", "end": "..."}

# Delete event by ID
{"action": "delete", "event_id": 1}
